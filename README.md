# Clone the repository 
```sh
  $ git clone https://github.com/JimmyBastos/RestWorkshop.git
```
# Runnig the API 
```sh
  $ cd Node-API
  $ yarn 
  $ yarn dev
```
# Runnig the Front-end 
```sh
  $ cd Web/vue-contacts
  $ yarn 
  $ yarn dev
```
