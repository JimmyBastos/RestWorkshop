# Clone the repository 
```sh
  $ git clone https://github.com/JimmyBastos/RestWorkshop.git
```
# Runnig the Back-end
## Install Dependencies 
```sh
  $ cd Node-API
  $ yarn
```
## API dev server
```sh
  $ yarn dev
```
## TESTS
```sh
  $ yarn test
```

# Runnig the Front-end 
```sh
  $ cd Web/vue-contacts
  $ yarn 
  $ yarn dev
```
